# 🍴 Food Party App
A React app to plan your party menu by selecting dishes, filtering categories, and tracking items.

## ✨ Features
- 🔍 Search dishes by name
- 🥗 Filter Veg/Non-Veg
- ➕ Add/Remove items from party menu
- 📊 Track selected items by category

Technologies----->
> React
> Vite
> CSS
> Html
> React Icons
> JSX
